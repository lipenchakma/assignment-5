What is Document Object Model (DOM)?

The DOM (Document Object Model) is a hierarchical tree structure representing a webpageâ€™s HTML elements. It allows JavaScript to access, modify, and interact with these elements dynamically. Developers use the DOM to change content, styles, and structure, enabling interactive and responsive web pages based on user interactions.

what do you select an element from the DOM?

To select an element from the DOM, use methods like getElementById() for an element by ID, getElementsByClassName() for elements by class, or querySelector() to select the first matching element with a CSS selector. These allow JavaScript to access and manipulate DOM elements dynamically.

What is event delegation in the context of the DOM, and why is it usefull?

Event delegation is a technique where a single event listener is attached to a parent element instead of multiple child elements. When an event occurs on a child, it "bubbles up" to the parent, which handles it. This is useful for improving performance by reducing the number of event listeners, simplifying code, and allowing dynamic elements to be managed without needing to reattach listeners for new child elements added later.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles in the DOM, you can use JavaScript methods like setAttribute() to set or update an attribute, and getAttribute() to retrieve the current value of an attribute. For styling, use the style property, like element.style.color to change text color. You can also directly set other CSS properties, such as background, width, or font size.
